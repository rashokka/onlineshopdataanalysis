Какие товары покупают чаще всего?
Какие товары приносят больше всего денег?
Какая страна приносит наибольшую выручку?
Как меняются продажи по месяцам?
В какие дни недели покупают чаще всего?
В какое время суток клиенты наиболее активны?
Какие суммы заказов встречаются чаще всего?
Каков диапазон цен и есть ли выбросы?
В какие сочетания дня и часа происходит больше всего продаж?
Как количество товара в заказе влияет на сумму заказа?
# Графики и коды
1.Сначала мы запускаем сервер pandas as pd, для работы с таблицами, внедряем датасет через переменную df (df = pd.read_csv) и выводим таблицу через код df.head()
2.С помощью df.info() я посмотрела количество строк и столбцов, типы данных и наличие пропусков
3.С помощью df.describe() я получила статистическое описание числовых колонок
4. Смотрим сколько уникальных знчений в каждом столбце через код df.nunique()
5.Я применила df.drop_duplicates(), чтобы убрать повторяющиеся строки
6.df.dropna() удалил строки с пропущенными знаками как пробелы итд 
7.Я преобразовала столбец с датой в формат datetime с помощью pd.to_datetime()
8.Выводим таблицу через df.head()
9.внедряем matplotlib.pyplot как библиотеку данных и seaborn для красивой визуализации.
задаем стиль ggplot.
