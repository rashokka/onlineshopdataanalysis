# Бизнес вопросы
Какие товары покупают чаще всего?
Какие товары приносят больше всего денег?
Какая страна приносит наибольшую выручку?
Как меняются продажи по месяцам?
В какие дни недели покупают чаще всего?
В какое время суток клиенты наиболее активны?
Какие суммы заказов встречаются чаще всего?
Каков диапазон цен и есть ли выбросы?
В какие сочетания дня и часа происходит больше всего продаж?
Как количество товара в заказе влияет на сумму заказа?
